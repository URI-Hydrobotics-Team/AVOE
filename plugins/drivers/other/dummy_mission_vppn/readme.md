Here lies Mission VPPN's dummy driver. The bane of all that is holy.

Mission VPPN's dummy driver is another plugin addition to the virtual suite for testing without a physical bot.
This plugins purpose is to give randomized fake 'CV' data for testing Mission_Vppn without a CV setup.
This plugin is paired with URI Hydrobotics CV repositories data output. The formating can be changed easily for your own purposes. 

Default format: name|confidence|time stamp|screen x offset|screen y offset|actual distance in z
